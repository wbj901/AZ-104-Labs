## ğŸ”§ Tasks Performed

### Task 1: Deploy an Azure Container Instance
1. Created a new **Azure Container Instance** using a **Quickstart Docker image** (`aci-helloworld`).  
2. Configured the container with a **public DNS name label** to enable browser-based access.  
3. Disabled container diagnostics logging during deployment to reduce overhead.  
4. Reviewed and validated deployment settings before provisioning the container instance.  

### Task 2: Test and Verify Deployment
1. Verified that the container instance status was **Running** in the Azure Portal.  
2. Accessed the containerized web application via its **FQDN** over port 80.  
3. Refreshed the application multiple times to generate HTTP request logs.  
4. Reviewed container logs to confirm successful GET requests from the browser.
